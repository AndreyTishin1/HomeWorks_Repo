# Туториал по работе с Git

## Начало работы

Для начала работы, необходимо инициализировать сам Git

Для его инициализации введите команду 

```
  git init
```

## Добавление файла

Для добавления файла в Git необходимо воспользоваться командой 

```
git add название файла
```

## Получение изменений и слияние с локальной версией производит команда:
```
git pull
```

## Отправляет локалную копию репозитория на внешний команда:
```
git push
```

## Показывает все установленные удаленные соединения команда:
```
git remote -v
```
